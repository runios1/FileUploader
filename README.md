# FileUploader

FileUploader is a modern web application that makes managing and sharing your files simple, secure, and intuitive. Whether you want to upload documents, organize content into directories, or access your data from anywhere, this app provides a seamless experience with a clean, user-friendly interface. Designed with real-world usability in mind, FileUploader leverages cloud storage to keep your files safe and easily accessible, while robust authentication ensures your data stays private. This project highlights a full-stack solution demonstrating practical skills in building secure and maintainable web applications.

## Tech Stack

**Frontend**: React, React Router, Vite (build tool)

**Backend**: Node.js, Express.js, Prisma ORM, PostgreSQL

**Storage**: Supabase Storage for file hosting

**Authentication**: JWT stored in secure, HTTP-only cookies with CORS support

**Deployment**:

  - **Backend**: Render Web Service and Neon database hosting
  
  - **Frontend**: Netlify

Designed with the help of Figma Make.
